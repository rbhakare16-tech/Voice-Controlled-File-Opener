🎙️ Voice Controlled File Opener

📌 Project Description

This project is a Voice-Controlled Desktop Automation System developed using Python.
It allows users to open folders and applications on their computer using voice commands.

The system listens to the user's voice, converts it into text, and performs the required action like opening Documents, Downloads, or Chrome.

---

🚀 Features

- 🎤 Voice recognition
- 📂 Open system folders (Documents, Downloads)
- 🌐 Open applications (Google Chrome)
- 🔊 Text-to-speech response
- 🧠 Simple AI-based command detection

---

🛠️ Technologies Used

- Python
- SpeechRecognition
- pyttsx3
- OS module

---

📂 How It Works

1. The system asks: "Which folder do you want to open?"
2. User gives voice input.
3. Speech is converted into text using Google Speech API.
4. The program checks the command using conditional statements.
5. The requested folder/application opens automatically.
